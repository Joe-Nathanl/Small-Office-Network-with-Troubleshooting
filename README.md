# Network Troubleshooting Lab – Small Office Network

## 📌 Overview

This lab simulates a real-world small office network environment where I configured, tested, and troubleshot common network issues using Cisco Packet Tracer.

The goal of this lab was to build a functional network and then simulate real-world problems such as DHCP failures, incorrect IP configurations, and interface issues, followed by diagnosing and resolving them using a structured troubleshooting approach.

---

## 🖥️ Network Topology

- 1 Router (Cisco 1941)
- 1 Switch (Cisco 2960)
- 2 PCs
- 1 Server (DHCP)
